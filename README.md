# Task Manager Web App

A full-stack Task Manager web application with complete CRUD functionality.

## Features
- User Registration and Login
- Add Tasks
- View Tasks
- Edit/Update Tasks
- Delete Tasks
- SQLite Database

## Tech Stack
- Python
- Flask
- HTML
- CSS
- SQLite

## How to Run
1. Run: python app.py
2. Open: http://127.0.0.1:5000/

## Project Description
This application allows users to manage their daily tasks. Users can register, login, add tasks, update tasks, and delete tasks. Data is stored using SQLite database.
